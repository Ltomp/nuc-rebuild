Scope

Three machines, one clean rebuild: NUC 14 Pro (Proxmox host, workstation VM + LLM LXC), Raspberry Pi 5 (bare-metal Home Assistant), HP Pro Mini 400 G9 (Proxmox host, Frigate LXC). Each phase below has a self-contained prompt in prompts/ — don't start the next phase until the current one's exit criteria are met. Checkbox state is saved in this browser only.

Deferred / out of scope

  • NAS build / Proxmox Backup Server on the HP Mini's free SATA bay — deferred in favor of LLM performance; vzdump-on-local-disk is the interim safety net.
  • Google Coral Dual Edge TPU M.2 accelerator — no compatible slot on any of the three machines (NUC/HP Mini are slotless SFF; Pi 5's HAT breakout is M-key, not E-key).
  • NPU detection experiment (OpenVINO NPU device on the NUC's Meteor Lake NPU) — revisit after the GPU baseline is proven.
